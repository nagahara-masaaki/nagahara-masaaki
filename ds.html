Sparse Representation of Feedback Filters in Delta-Sigma Modulators

MATLAB Codes

Requirements
  • You need to install YALMIP
    https://yalmip.github.io/<
    and SDPT3
    https://github.com/SQLP/SDPT3
  • We checked the code on MATLAB 2017b.
MATLAB function
  • NTF_MINMAX_sparse_L1.m: main function
Example
>ord = 64; %FIR filter order of the proposed filter
>Om = pi/32; %Cut-off frequency
>Hf=1.5; %H-infinity norm of NTF
>lam = 10; %Regularization parameter
>th = 1e-4; %Thresholding parameter
>[ntf, F] = NTF_MINMAX_sparse_L1(lam,ord,Om,Hf,1,th);

By running the above codes, you get ntf, the transfer function of NTF, and F, the filter coefficient of F(z).
